# Vehicle & Human Counter System

A computer vision based vehicle and human counting system developed using YOLOv8 and ByteTrack.

## Features

- Human detection
- Vehicle detection
- Multi-object tracking
- ROI based counting
- CPU optimized processing
- Tkinter GUI
- Real-time counting
- Different detection modes

## Technologies

- Python
- OpenCV
- YOLOv8
- ByteTrack
- Tkinter
- Supervision
